About

This beginner's tutorial was created by Merlin Rebrović for the Hyde project. It is included in Hyde as a layout named starter. If you have a default Hyde installation, all you have to do is write:

hyde -s folder_name create -l starter

To check for the newest version, to download it separately or to report a bug, please visit Hyde Starter Kit's GitHub page.
